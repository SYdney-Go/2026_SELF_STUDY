## 3.2 연산자의 연산 방법
### 3.2.1 산술 연산자와 증감 연산자
＊ 아는거임 패스
### 3.2.2 비트 연산자
기호만 까먹지 말자. 진법 표기는 그냥 그때 그때 필요하면 줍줍하자.
- AND 연산 :  &
- OR 연산 : | 
- XOR 연산 : ^
- NOT 연산 : ~

```
int value1 = 3;
System.out.println(false && ++value1 > 6); //false
System.out.println(value1); //3 (숏 서킷으로 ++value1동작 X)

int value2 = 3;
System.out.println(false & ++value2 > 6); //false
System.out.println(value2); //4
```

### 3.2.3 시프트 연산자
시프트 연산자도 그때 그때 필요하면 찾자

### 3.2.4 비교 연산자
이건 기억해야지. 언어마다 쬐끔 다름
비교의 대상은 스택 메모리에 있는 값을 비교한다.
```
//@ 참조자료형 등가비교 (번지비교)

String str1 = new String("안녕");

String str2 = new String("안녕");

System.out.println(str1 == str2); //false
```

### 3.2.5 논리 연산자
비트 연산자랑 헷갈리지만 말자
- AND 연산 :  &&
- OR 연산 : || 
- XOR 연산 :  (XOR은 쇼트 서킷을 쓸 수 없어서 기호가 같음)
- NOT 연산 : !
