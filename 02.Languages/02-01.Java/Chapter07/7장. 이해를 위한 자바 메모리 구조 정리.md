```
자바 메모리 = 클래스 영역 + 스택 영역 + 힙 영역
```

## 1. 클래스 영역
`프로그램 실행에 필요한 클래스 정보 저장`

- 클래스 이름
- 필드 정보
- 메서드 정보 
- static 변수
- 상수 정보

## 2. 스택 영역
`메서드가 실행되는 동안 필요한 값들이 저장. 실행이 끝나면 자동으로 사라진다.`

- 지역변수
	- 특징 1. 스택의 지역변수는 자동 초기화 되지 않는다
	- 특징 2. 실행이 끝나면 자동 삭제된다.
- 매개 변수
- 메서드 호출 정보
- 참조 변수

## 3. 힙 영역
`new 로 생성한 객체와 배열 정보 저장. 스택 영역의 참조 변수 속 주소가 가리키는 실제 객체 대상`

- 특징 : 힙의 객체와 배열들은 자동으로 기본값을 가진다.
