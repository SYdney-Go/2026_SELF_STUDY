# 2026_SELF_STUDY

AX/DX, 데이터 처리, 백엔드, 시스템 아키텍처에 대한 학습 기록입니다.

2023년 이후 실무를 하며 데이터가 생성되고, 흐르고, 처리되고, 서비스와 운영 시스템으로 연결되는 구조에 관심이 커졌습니다.

이 저장소에는 학습한 개념, 실습 코드, Markdown 정리 문서를 기록합니다.

데이터와 시스템 구조를 이해하고,
