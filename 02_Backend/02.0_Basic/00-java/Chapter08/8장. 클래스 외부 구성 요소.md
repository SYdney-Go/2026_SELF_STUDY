## 8.1 패키지와 임포트
```
// 외부 구성요소 순서
패키지
임포트
외부클래스
```
### 8.1.1 패키지
비슷한 목적으로 생성된 클래스 파일을 한 곳에 모아둔 폴더
### 8.1.2 임포트
다른 패키지 내의 클래스를 사용하기 위한 문법 요소
- 방법 1. 클래스의 풀네임 사용하기
  (예) `패키지명.클래스명`
  이때 다른 패키지의 클래스를 사용하려면 public class로 선언되어야 한다.
- 방법 2. 임포트 사용
  (예) `import 패키지명.클래스명`
  이때 다른 패키지에 있는 같은 이름의 클래스를 중복해서 사용할 수 없다. 이럴떄 1번 방법과 혼용해서 쓰면 된다.

## 8.2 외부 클래스
public 클래스의 외부에 추가로 정의한 클래스
하나의 java 파일에서는 하나의 public class만이 존재할 수 있으므로 외부에서 사용해야하는 클래스는 따로 파일을 만들어서 분리하자.
