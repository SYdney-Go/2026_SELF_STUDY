# 범위 2: 작은 API에서 관제 시스템으로

> 아직 작성 전. v1 API 완성 후 같이 채웁니다.

## 확장 경로 (미리보기)

```text
v1  Event CRUD API
      ↓
    검색 / 필터 / 페이지네이션
      ↓
    사용자 인증 / 관리자 권한
      ↓
    실시간 알림 (WebSocket, 메시지 큐)
      ↓
    이벤트 통계 / 대시보드 API
      ↓
    관제 시스템 아키텍처
```

지금은 v1만 보면 됩니다. 이 문서는 "이게 어디로 가는지" 맥락용입니다.
